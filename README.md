# 🌸 Digital Diary & Pomodoro Hub

"Not me, at 2 AM, trying to build my own personalized Pomodoro web."

### 🔗 Live Demo
Check out my live project here: [https://custom-pomodoro-taupe.vercel.app/](https://custom-pomodoro-taupe.vercel.app/)

### 📖 The Story Behind This Project
I’ve tried so many productivity tools—YPT, Pomofocus, Flip, you name it. They are all incredibly well-made and useful. But somehow, I could never find one that *perfectly* fits my daily workflow, specifically for my chaotic, high-energy student life. 

I needed something that isn't just a timer, but a companion. I wanted a space that tracks my tasks based on my actual urgency, categorizes my crazy schedule (from *Matkul* to *Organisasi* to *Lomba*), and keeps my focus on track with a built-in Pomodoro. Something that doesn't just spread my tasks like a standard calendar, but keeps them organized in a clean, minimalist timeline right next to my focus area.

So, I decided to stop searching and start building. This is my own little corner for getting things done, designed by me, for me.

### ✨ Features
*   **Dynamic Pomodoro Timer**: A 25-minute focus session (with 3 sets) that automatically syncs with your most urgent task.
*   **Smart Timeline**: A dedicated sidebar showing your tasks ordered by the nearest deadline, so you never miss a beat.
*   **Activity Hub**: A place to track what I'm currently involved in (Committees, Competitions, etc.).
*   **Aesthetic & Minimalist**: Designed with my favorite "cute girl" aesthetic to keep me motivated and happy while working.
*   **Personalized Workflow**: Because I know exactly what I need to do next, I can manually select tasks to focus on at any time.

### 🛠️ Built With
*   **React.js** - For building the dynamic UI.
*   **Vite** - For a lightning-fast development experience.
*   **Tailwind CSS** - To make it look beautiful and perfectly responsive.
*   **Vercel** - For seamless deployment.

### 🚀 Getting Started
1. Clone this repository.
2. Run `npm install` to get all the necessary packages.
3. Run `npm run dev` to start your own productivity journey.
   
📝 Project Note
Please Note: This is a passion project built as a personalized "digital companion." For the sake of simplicity and immediate usability, this application currently does not have a backend or database
